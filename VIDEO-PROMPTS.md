## TIPS FOR BEST RESULTS

### Nano Banana:
- Add "photorealistic, 8K, cinematic" to every prompt
- Specify "Utah" and "Wasatch" for geographic accuracy
- Use "architectural photography" or "drone photography" for the right lens feel
- Avoid people's faces in close-up (AI struggles with faces)

### Higgsfield:
- Always specify "seamless loop" for website background videos
- Keep camera motion minimal — slow zoom, slow pan, slow orbit
- Specify "5-8 second" duration
- Add "steady, smooth, no jitter" to prevent shaky output
- Use the Nano Banana image as the input/reference frame
- For weather effects (rain, wind, clouds), describe the motion explicitly

### Post-processing:
- Compress all videos with: `ffmpeg -i input.mp4 -vf "scale=1280:-2" -c:v libx264 -crf 28 -preset slow -an -movflags +faststart output.mp4`
- Target file size: under 1MB per video for web performance
- Extract first frame as poster: `ffmpeg -i input.mp4 -vframes 1 -q:v 2 poster.jpg`
- All videos should be muted (no audio track) for autoplay compatibility
